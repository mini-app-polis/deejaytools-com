# Architecture Decision Records — deejaytools-com-app

This directory contains Architecture Decision Records (ADRs) for the
React web application. ADRs document significant architectural
decisions, the context around them, and their consequences.

## Format

Each ADR is a markdown file named `ADR-NNN-title-in-kebab-case.md`
where `NNN` is a zero-padded 3-digit sequence number starting at `001`.
This matches ecosystem-standards DOC-005.
